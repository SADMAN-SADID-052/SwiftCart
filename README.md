## Question Answer


#### 1) What is the difference between `null` and `undefined`?

#### Answer:

 null হলো কোনো মান deliberately খালি রাখা। যেমন: let a = null; মানে intentional খালি।
undefined হলো ভ্যারিয়েবল ডিক্লেয়ার করা আছে কিন্তু এখনো কোনো মান অ্যাসাইন করা হয়নি। যেমন: let b; console.log(b); // undefined

#### 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?

#### Answer: 

map() হলো array এর প্রতিটি element এ operation করে নতুন array return করে।

forEach() ও array এর প্রতিটি element এ operation চালায়, কিন্তু কোনো array return করে না।

#### 3) What is the difference between `==` and `===`?

#### Answer: 

== চেক করে শুধু value মিলছে কি না, type মিলানোর কথা ভাবে না।
=== চেক করে value ও type দুটোই এক কিনা।

#### 4) What is the significance of `async`/`await` in fetching API data?

#### Answer: 

async/await দিয়ে আমরা API থেকে data fetch করি যেন কোডটা synchronous এর মতো readable হয়। এতে .then() চেইন দরকার হয় না, error handle করাও সহজ।

#### 5) Explain the concept of Scope in JavaScript (Global, Function, Block).

#### Answer: 

Global Scope: ভ্যারিয়েবল যেখানে declare হয়েছে, পুরো কোডে accessable।

Function Scope: ভ্যারিয়েবল শুধুমাত্র সেই function এর ভেতর কাজ করবে।

Block Scope: { } এর ভেতরে declare করা let বা const শুধু ওই block এ valid, বাইরে access নেই।
